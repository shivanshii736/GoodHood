# CommunityConnect (Tkinter App)

CommunityConnect is a Python desktop app built using Tkinter & SQLite.
It allows users to:

- Create community service events
- View all ongoing events
- Store events locally using SQLite database

This is a beginner-friendly team project for learning GUI development and Python collaboration using GitHub.

## How to run

1. Install Python (3.8+)
2. Install dependencies:
	pip install -r requirements.txt
3. Run:
	python main.py
